# Animation Library

A collection of modern, smooth, and performance-optimized animation effects for web applications.

## Live Previews

**[Fade Animation](./fade/index.htm)** — Simple fade and text split animations using GSAP library. Elements fade in when they appear in viewport, text splits into words with staggered animation timing.

**[Odometer Counter](./odometer-counter/index.html)** — Smooth scroll-triggered counter animation with rolling odometer-style numbers. Perfect for displaying statistics that animate when entering the viewport.

**[Scroll Snap Slider](./scroll-snap-slider/index.html)** — Scroll-based slider with pinned sections and smooth image transitions. Uses GSAP ScrollTrigger for precise scroll-driven animations with swipe/keyboard navigation support.
